# islandium-regions

Gestion des régions et protections

## 📦 Installation

Ce module fait partie du projet [Islandium](https://github.com/So-Hytale), un serveur Hytale.

### Build

```bash
./gradlew :islandium-regions:build
```

### Dépendances

Voir `build.gradle` pour la liste complète des dépendances.
